tips

pure CSS tooltips because tooltips should be simple

I've got a tooltip
Hover over me or tap me!

tips work on :hover and can be positioned using helper classes and tweaked using less/sass sources.

Due to the way :hover behaves on mobile devices, this means that they will appear on tap.

Alternatively, you can show/hide tips by toggling the class show

  1. Include the stylesheet
  2. Add the attribute data-tip to an element that you wish to have a tooltip
  3. Use the attribute data-tip-content to declare the tooltip. content
  4. Lastly add any helper classes for positioning or sizing.
